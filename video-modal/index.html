What if you had a video modal, but it's pretty fast?

This heavily cribs from Paul Irish's Lite YouTube embed so credit where it's entirely due.

Basic Usage

Include video-modal.js and video-modal.css and then instantiate a basic video modal like so.

<video-modal video="Aj6SF0Bek_E" title="NEW DGD HOMIE"></video-modal>

Inline Button Usage

Add a <video-modal> element as you normally would, but add the 'button' attribute to prevent automatic thumbnail rendering, like so: <video-modal button>. Then simply add the markup of your choosing inside the <video-modal> tag.
